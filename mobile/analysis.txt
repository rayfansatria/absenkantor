=== Flutter Project Analysis ===

This is a complete Flutter mobile application for office attendance management.

STRUCTURE:
- Clean Architecture pattern with separation of concerns
- BLoC pattern for state management  
- Repository pattern for data access
- Service locator (get_it) for dependency injection

KEY COMPONENTS:
1. Authentication: Complete login/logout with JWT
2. Dashboard: Employee portal with quick access to features
3. Attendance: GPS-validated clock in/out with photo capture
4. Leave Management: Request and track leave applications
5. Overtime: Overtime request management
6. Schedule: View work shifts and holidays
7. Payroll: Access payslips
8. Profile: User profile management

DEPENDENCIES:
- flutter_bloc: State management
- dio: HTTP client
- shared_preferences: Local storage
- local_auth: Biometric authentication
- camera: Photo capture
- geolocator: GPS location
- google_maps_flutter: Maps integration
